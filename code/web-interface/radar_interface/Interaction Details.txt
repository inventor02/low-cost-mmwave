----
DCA1000EVM Control fpga overwatch.json

DCA1000EVM Control start_record overwatch.json

mmwaveexample.exe

DCA1000EVM Control fpga imaging.json

DCA1000EVM Control start_record imaging.json

mmwaveexample.exe

mmwaveconfig.txt (one for each, need to be in seperate folders)
